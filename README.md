# 🛒 Supermarket Customer Dashboard

## 📊 Overview
Interactive Excel dashboard analyzing customer satisfaction, shopping behavior, and store performance.

## 🔍 Key Insights
- Longer queue times reduce customer satisfaction
- Loyalty card users show higher satisfaction
- Peak visit times impact shopping duration

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- Data Visualization
